###Features

- Support SQLAlchemy models
- Model generator
- Databases: PostgreSQL

# Flask Headless CMS

###Setup

####Using python

    `$ python create_app.py <new_project> -s skeleton -g`

####Using alias

    `$ alias flaskcli="python /path/to/startup/script/create_app.py"`

    `$ flaskcli <new_project> -s skeleton -g`
